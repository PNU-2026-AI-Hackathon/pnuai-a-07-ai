# 사고확률추정기 — 베이즈 정리 기반 산재 발생 위험도 추정 모듈

## 역할

근로자의 특성(산업, 연령, 성별, 지역, 사업장 규모)을 입력하면  
**어떤 유형의 사고가 얼마나 높은 확률로 발생하는지** 를 순위별로 반환한다.

---

## 원리

### 왜 단순 KOSHA 비율이 아닌가?

KOSHA 마이크로데이터는 **사고가 이미 난 사람들의 기록**만 있다.  
예: "건설업 40대 남성이 산재가 많다" → 단순히 건설업에 40대 남성이 많아서일 수도 있음.

이 편향을 보정하기 위해 **베이즈 정리**를 적용한다.

### 수식

$$P(\text{발생형태}=k,\ \text{사고} \mid F) \propto \frac{P(F,\ k \mid \text{사고}) \times P(\text{사고})}{P(F)}$$

| 항 | 의미 | 데이터 파일 |
|---|---|---|
| $P(F, k \mid \text{사고})$ | 산재 발생자 중 특성 F이고 발생형태가 k인 비율 | `P_FL_given_accident.csv` |
| $P(\text{사고})$ | 전체 근로자 대비 전체 재해자 수 (상수, 전 업종 합산) | `P_업종사고_재해율.csv` |
| $P(F)$ | 전체 근로자 중 특성 F인 비율 (IPF 추정) | `P_F_joint.csv` |

---

## 입력 피처

`predict()` 메서드에 아래 5개 인자를 전달한다.

### 1. `산업` — 업종 (str)

| 값 |
|----|
| `'건설업'` |
| `'광업'` |
| `'금융및보험업'` |
| `'기타의사업'` |
| `'농업'` |
| `'어업'` |
| `'운수·창고·통신업'` |
| `'임업'` |
| `'전기·가스·증기·수도사업'` |
| `'제조업'` |

### 2. `연령` — 연령대 (str)

KOSHA 원본의 세분화된 연령을 아래 **5개 표준 그룹**으로 입력한다.

| 입력값 | 해당 연령 |
|--------|---------|
| `'20~29세'` | 18세 미만 + 18~24세 + 25~29세 |
| `'30~39세'` | 30~34세 + 35~39세 |
| `'40~49세'` | 40~44세 + 45~49세 |
| `'50~59세'` | 50~54세 + 55~59세 |
| `'60세이상'` | 60세 이상 |

### 3. `성별` — 성별 (str)

| 값 |
|----|
| `'남'` |
| `'여'` |

### 4. `시도` — 지역 (str)

| 값 (16개 시도) |
|----------------|
| `'강원'`, `'경기'`, `'경남'`, `'경북'`, `'광주'`, `'대구'`, `'대전'`, `'부산'` |
| `'서울'`, `'울산'`, `'인천'`, `'전남'`, `'전북'`, `'제주'`, `'충남'`, `'충북'` |

> 세종시는 KOSHA 마이크로데이터에 산재 케이스 없음 → 지원 안 됨

### 5. `규모` — 사업장 규모 (str)

| 값 | 종업원 수 |
|----|----------|
| `'5인 미만'` | ~4명 |
| `'5~9인'` | 5~9명 |
| `'10~19인'` | 10~19명 |
| `'20~49인'` | 20~49명 |
| `'50~99인'` | 50~99명 |
| `'100~299인'` | 100~299명 |
| `'300~499인'` | 300~499명 |
| `'500~999인'` | 500~999명 |
| `'1,000인 이상'` | 1,000명 이상 |

---

## 출력

`pandas.DataFrame` 반환. 컬럼:

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `발생형태` | str | 사고 유형 (넘어짐, 떨어짐, 끼임 등 최대 23개) |
| `P_risk` | float | 해당 발생형태의 사고 위험도 점수 |
| `순위` | int | 위험도 높은 순 (1 = 가장 위험) |

> `P_risk`는 절대 확률이 아닌 **상대적 위험도 점수**이며, 특성 조합 간 비교에 사용한다.

---

## 사용 예시

```python
from risk_estimator import RiskEstimator

est = RiskEstimator()  # 데이터 한 번만 로드

result = est.predict(
    산업='건설업',
    연령='40~49세',
    성별='남',
    시도='경기',
    규모='50~99인',
)

print(result)
#      발생형태   P_risk  순위
#       떨어짐 0.007101   1
#     물체에맞음 0.003842   2
#       넘어짐 0.003705   3
#        끼임 0.002881   4
#       부딪힘 0.002401   5
#    절단베임찔림 0.001749   6
#     업무상질병 0.001235   7
#    깔림.뒤집힘 0.000995   8
# 불균형및무리한동작 0.000960   9
#       무너짐 0.000446  10
#       ...

# 상위 3개만 보기
print(result.head(3))

# 1위 사고 유형
top = result.iloc[0]
print(f"가장 위험한 사고: {top['발생형태']} (위험도 {top['P_risk']:.4f})")
```

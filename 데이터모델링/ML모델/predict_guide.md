# predict.py 사용 가이드

LightGBM 학습 모델을 기반으로 산재 사고의 발생형태·재해정도·질병종류·세부질병종류 확률을 Top-K로 반환하는 예측 모듈.

---

## 입력 파라미터

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|:----:|------|
| `대업종` | str | ✓ | 아래 허용값 참고 |
| `종업종` | str | ✓ | KOSHA 원본 문자열 (내부에서 자동 정규화) |
| `성별` | str | ✓ | `남` / `여` |
| `연령` | str | ✓ | 아래 허용값 참고 |
| `근무기간` | str | ✓ | 아래 허용값 참고 |
| `규모` | str | ✓ | 아래 허용값 참고 |
| `지역` | str | ✓ | 아래 허용값 참고 |
| `건설공사금액` | str \| None | - | 대업종이 `건설업`일 때만 유효. 생략 시 `해당없음` 처리 |
| `년도` | int | - | 예측 기준 연도 (기본값: `2024`) |
| `top_k` | int | - | 반환할 후보 수 (기본값: `3`) |

### 허용값 목록

**대업종**
```
건설업 / 광업 / 기타의사업 / 운수·창고·통신업 / 제조업
농업 / 어업 / 임업 / 전기·가스·증기·수도사업 / 금융및보험업
```
> 농업·어업·임업·전기·가스·증기·수도사업·금융및보험업은 소규모 통합 모델 사용

**연령**
```
~19세 / 20세~24세 / 25세~29세 / 30세~34세 / 35세~39세
40세~44세 / 45세~49세 / 50세~54세 / 55세~59세 / 60세~64세 / 65세~
```

**근무기간**
```
1개월 미만 / 1~3개월 미만 / 3~6개월 미만 / 6개월~1년 미만
1~2년 미만 / 2~3년 미만 / 3~4년 미만 / 4~5년 미만 / 5년 이상
```

**규모**
```
1인 / 2~4인 / 5~9인 / 10~29인 / 30~49인 / 50~99인 / 100~299인 / 300인 이상
```

**지역**
```
서울 / 부산 / 대구 / 인천 / 광주 / 대전 / 울산 / 경기 / 강원
충남 / 충북 / 전남 / 전북 / 경남 / 경북 / 제주
```

**건설공사금액** (건설업 전용)
```
해당없음 / 3억 미만 / 3억~5억 미만 / 5억~10억 미만 / 10억~30억 미만
30억~50억 미만 / 50억~100억 미만 / 100억~300억 미만 / 300억 이상
```

---

## 반환값

```python
{
    '발생형태':    list[tuple[str, float]] | None,
    '재해정도':    list[tuple[str, float]] | None,
    '질병종류':    list[tuple[str, float]] | None,
    '세부질병종류': list[tuple[str, float]] | None,
}
```

- 각 항목은 `(라벨, 확률)` 튜플 리스트로 확률 내림차순 정렬
- 해당 업종의 모델 파일이 없으면 `None` 반환

## 사용 예시

### 1. Python import

```python
from predict import predict

result = predict(
    대업종   = '제조업',
    종업종   = '기계기구·금속·비금속광물제품제조업',
    성별     = '남',
    연령     = '40세~44세',
    근무기간 = '1~2년 미만',
    규모     = '5~9인',
    지역     = '경기',
    top_k    = 3,
)

print(result['발생형태'])
# [('끼임', 0.2878), ('물체에맞음', 0.1225), ('절단베임찔림', 0.1152)]

print(result['재해정도'])
# [('91~180일', 0.4044), ('29~90일', 0.3414), ('6개월 이상', 0.1755)]

print(result['질병종류'])
# [('요통', 0.4936), ('신체부담작업', 0.3628), ('뇌.심혈관질환', 0.0878)]

print(result['세부질병종류'])
# [('신체부담작업', 0.3568), ('사고성 요통', 0.3044), ('비사고성 요통', 0.1887)]
```

### 2. 터미널 CLI

```bash
python predict.py \
  --대업종 제조업 \
  --종업종 "기계기구·금속·비금속광물제품제조업" \
  --성별 남 \
  --연령 "40세~44세" \
  --근무기간 "1~2년 미만" \
  --규모 "5~9인" \
  --지역 경기 \
  --top_k 3
```

건설업 (건설공사금액 포함):
```bash
python predict.py \
  --대업종 건설업 \
  --종업종 건축공사업 \
  --성별 남 \
  --연령 "50세~54세" \
  --근무기간 "1개월 미만" \
  --규모 "5~9인" \
  --지역 서울 \
  --건설공사금액 "5억~10억 미만"
```

## 내부 동작

```
입력값
  └─ 대업종 판별 → 모델 파일 suffix 결정
       ├─ 소규모(농업/어업/임업/전기가스/금융) → 소규모통합 모델
       └─ 대규모(건설/광업/기타/운수/제조)     → 업종별 개별 모델

  └─ 종업종 정규화 (SUBJOB_NORM: 82개 원본 → 44개 통합)
  └─ 전체 피처 인코딩 (_build_row)
       ├─ 규모/연령/근무기간/종업종 → 순서형 정수 인코딩
       ├─ 성별/지역 → 원핫 인코딩
       └─ 건설공사금액 → 건설업 전용 순서형 인코딩

  └─ 모델별 predict → softmax 확률 배열
  └─ Top-K 추출 → 역매핑(enc → 한글 라벨)
  └─ dict 반환
```

**모델 파일 위치**: `ML모델/models/{태스크명}_{업종명}.txt`

| 태스크명 | 예측 대상 |
|----------|----------|
| `발생형태` | 사고 유형 (23개 클래스) |
| `재해정도_발생형태기반` | 재해 심각도 (8개 클래스) |
| `질병종류` | 업무상질병 유형 (7~9개 클래스) |
| `세부질병종류` | 세부 질병 분류 (21~32개 클래스) |

---

## 주의사항

- `predict.py`는 `ML모델/` 폴더 기준으로 실행하거나, `sys.path`에 해당 경로가 포함되어 있어야 함
- `종업종`은 KOSHA 원본 문자열을 그대로 넣으면 내부에서 자동 정규화 처리됨
- `질병종류` / `세부질병종류`는 업무상질병 데이터만으로 학습된 모델 — 사고성 재해에 적용 시 참고용으로만 활용
